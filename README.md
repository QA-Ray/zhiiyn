# 知音 (Zhiyin)

懂你的 AI 音乐伴侣——根据你的口味、时段、心情推荐和续播，像一个懂音乐的朋友在身边。

## 下载

最新 macOS 安装包请访问 [Releases](https://github.com/QA-Ray/zhiiyn/releases/latest)。

支持的系统：macOS 11+ on Apple Silicon (M1/M2/M3/M4)

## 安装

1. 下载 `.dmg`，双击挂载
2. 把【知音.app】拖到【应用程序】文件夹
3. 从启动台 / 应用程序里双击打开

如果首次双击弹"无法打开"，在 .app 上**右键 → 打开**，再点一次"打开"即可（macOS Tahoe 首次启动联机校验偶尔会卡，跟应用本身安全性无关）。
